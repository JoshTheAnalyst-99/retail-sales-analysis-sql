# SQL Business Analysis Project

## Overview
This project contains SQL queries used to analyze sales, customers, products, employees, and transactions data.

## Skills Demonstrated
- Joins
- Aggregations
- Window Functions
- CTEs
- Business Analysis
- Data Filtering
- Revenue Analysis

## Key Business Questions Answered
1. Top-performing salesperson
2. Highest-selling products
3. Loyal customers
4. Monthly sales trends
5. Negative account balances
6. Unordered products
7. Employees earning above department average
8. Revenue by category

## Tools Used
- SQL Server
- T-SQL

## Dataset
Mock business datasets including:
- Orders
- Products
- Employees
- Transactions
- Sales
